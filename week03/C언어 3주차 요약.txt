변수의 이름을 정하는 방법: 

** 중요 
        식별자: 변수 이름은 프로그래머가 마음대로 정하지만 규칙을 지켜야함 -> 변수의 이름은 변수들을 식별하는 역할을함
          규칙1. 식별자는 영문자와 숫자, 밑줄문자 (_) 로 이루어진다.
          규칙2. 식별자 중간에 공백이 들어가면 안된다.
          규칙3. 첫글자는 반드시 영문자 or 밑줄 기호(_)여야한다. -> 숫자로 시작할 수 없음
          규칙4. 대소문자는 서로 구별된다. 따라서 index != Index != Index
          
        sum, _count, number_of_picture, King3 //모두 가능
        2nd_base, money샵, double // 모두 불가능
